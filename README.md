# Durian Vote Poll API

A FastAPI project for creating and managing a durian preference poll with MongoDB.

## Features
- Create poll data
- Read current poll results
- Vote for a durian preference option

## Tech Stack
- FastAPI
- MongoDB
- PyMongo
- Python-dotenv
- Pydantic

1. Clone the repository
2. Install dependencies
